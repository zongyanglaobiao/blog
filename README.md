# 介绍
**使用hugo搭建的博客，记录自己的学习笔记、学习心得、生活感悟**

## ➡️Hugo启动命令

- `hugo server --watch` 启动之后用于实时监控变化，方便调试
- `hugo server  --buildDrafts ` 用于是否显示草稿文章
- `hugo server  --theme ` 用于是否显示草稿文章
